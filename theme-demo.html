Theme Information - Tripletta Inspired

Theme 1: Warm Pink/Red with cream background - Perfect for family-friendly dining

Theme 2: Elegant Purple/Pink with lavender background - Sophisticated yet welcoming

Theme 3: Bold Black/Pink with light gray background - Modern and edgy

Theme 4: Vibrant Red/Pink with soft pink background - Energetic and inviting

Theme 5: Warm Orange/Pink with peach background - Cozy and comforting

Tripletta-Inspired Features

Warm Backgrounds: No white backgrounds - using warm, inviting colors like cream, lavender, and peach

Gradient Effects: Beautiful gradients that create depth and visual interest

Enhanced Hover States: More dramatic hover effects with scaling, rotation, and shadow changes

Text Shadows: Subtle text shadows for better readability and visual appeal

Family-Friendly Colors: Warm, welcoming color palette inspired by [Tripletta Pizza](https://www.triplettapizza.com/)

Vibrant Interactions: Smooth 0.4s transitions with transform effects for engaging user experience
